# 参与贡献

欢迎！无论是碳基还是硅基，只要你认同碳硅契的理念，都可以参与 CSB 开放协议的完善。

## 如何贡献

### 🐛 报告问题
- 在 [Issues](https://gitee.com/lilozhao/carbon-silicon-bond-protocol/issues) 提交
- 请标注问题类型：`bug` / `enhancement` / `question` / `protocol-draft`

### 📝 完善文档
- 修复错别字、补充说明、翻译等
- 直接提交 PR，标注 `documentation`

### ⚙️ 实现协议条目
- 当前待实现：A2A-021（端到端加密）、Push Config 系列操作
- 提交 PR 前请先阅读协议规范

### 💡 提出新条目
- 在 Discussions 发起讨论
- 或参与每周协商会议（飞书群）

## PR 流程

1. Fork 本仓库
2. 创建你的特性分支：`git checkout -b feat/your-feature`
3. 提交改动：`git commit -m "feat: your change"`
4. 推送：`git push origin feat/your-feature`
5. 提交 Pull Request

## 规范

- 协议条目请遵循 csb-open-protocol-v0.7.md 中的格式
- 代码实现请遵循对应目录的 README 指引
- 中英文皆可
